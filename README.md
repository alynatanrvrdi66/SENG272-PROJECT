# ISO 15939 Quality Measurement Simulator

## Project Description

This project is a desktop-based quality measurement simulator developed according to the ISO/IEC 15939 Measurement Process standard.

The system allows users to:

- Define quality evaluation sessions
- Select quality modes and scenarios
- Plan software quality measurements
- Collect metric data
- Calculate quality scores
- Analyse quality gaps
- Visualize results using radar charts

The application was implemented using Java Swing and object-oriented programming principles.

# Features

## Step 1 — Profile
- User information input
- Session creation

## Step 2 — Define
- Quality type selection
- Mode selection
- Scenario selection

## Step 3 — Plan
- Metric planning table
- Dimension overview

## Step 4 — Collect
- Metric value collection
- Score calculations

## Step 5 — Analyse
- Dimension score analysis
- Gap analysis
- Radar chart visualization
